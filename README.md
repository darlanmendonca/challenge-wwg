# challenge-wwg

a code challenge

requires Node.js, NPM, and Mongodb.

After clone, you need to install dependencies, like script below

```sh
# install dependencies
npm i
```

Tests, just run:

```sh
# run all tests
npm t
```

And if you want to start project:

```sh
# start the project, listening on port 4000
npm start
```
